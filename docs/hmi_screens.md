# HMI Screens

## HMI Overview

The first screen is the working dashboard. It contains start, stop, E-Stop, reset, auto/manual mode, conveyor and motor status, sensor states, counts, cycle time, throughput, alarm banner, active station, and operator instructions.

## Digital Twin View

The digital twin shows the conveyor, infeed sensor, inspection station, reject diverter, outfeed sensor, motor/drive block, PLC block, HMI/SCADA block, edge analytics block, and moving packages. Components are clickable and expose live tag values.

## Alarms

The alarm view separates active, acknowledged, and cleared alarms. Each record includes timestamp, severity, root cause hint, recovery steps, and effect on machine state.

## Trends

Trend cards show cycle time, throughput, OEE, reject rate, downtime, motor load/current, and sensor transitions.

## Maintenance

The maintenance view summarizes motor runtime, jam count, MTBF, MTTR, recommendations, fault history, and sensor health.

## Tag Browser

The tag browser supports search and quality filtering. It shows tag name, type, value, source, description, last changed timestamp, and quality.

## Manual Controls

Manual controls include jog conveyor, reject gate force, sensor force/release, pause/continue, simulation speed, and step-one-scan behavior.

## Production Report

The report screen summarizes counts, OEE, downtime reasons, quality results, and exports.
